# 🧭 VaultLite（最终 V1 项目方案）

> **Local-first Encrypted Password Vault + GitHub Auto Backup**

---

# 1. 项目本质（最重要）

VaultLite 是一个：

> **纯前端密码保险箱 + 本地加密 + GitHub 私有仓库自动备份工具**

它不是：

* ❌ 云服务
* ❌ 多端同步系统
* ❌ SaaS
* ❌ 账号系统平台

它只是：

> 一个浏览器里的加密工具 + 一个 GitHub 备份机制

---

# 2. 核心设计原则（不可变）

## Principle 1：本地优先（Local First）

* 所有数据先存在浏览器
* IndexedDB 是唯一运行态存储

---

## Principle 2：零信任后端

* GitHub 只存密文
* 不参与加解密
* 不知道主密码

---

## Principle 3：自动备份，不是同步

* 用户不操作备份
* 系统自动处理
* 但不会实时双向同步

---

## Principle 4：单文件数据模型

唯一数据文件：

```text
vault.enc
```

## 📌 触发机制（自动）

## ⏱ 防抖策略（关键）

```text

   ↓

   ↓
自动备份 GitHub
```

如果期间继续修改 → 重置计时

---

## 🧠 备份逻辑

```text
vault.json
   ↓

   ↓

   ↓
上传 GitHub
```

---

## 📦 GitHub 仓库结构

```text

└── vault.enc
```

---

## 🔁 每次备份行为

* 覆盖 `vault.enc`

* 不维护版本逻辑

# 🔐 6. 加密系统（核心安全）

---

## 加密流程

```text
Master Password
     ↓
PBKDF2 (SHA-256, 150k+ iterations)
     ↓
AES-256-GCM
     ↓
vault.enc
```

---

## 解密流程

```text
vault.enc
     ↓
AES-GCM decrypt
     ↓
JSON Vault
```

---

## 关键规则

* 主密码永不上传
* 密钥只存在内存
* 关闭页面即清空
